# Glossary

- **Coin**: The native currency on the <Glossary word="blockchain"></Glossary>.
- **Token**: Not the native currency on the <Glossary word="blockchain"></Glossary>. If the <Glossary word="blockchain"></Glossary> is “Ethereum”, then only “id”: “ETH” has the type “coin”. All the other currencies on the <Glossary word="Ethereum"></Glossary> network have the type “token”. Here is an example of USDC on the <Glossary word="ethereum"></Glossary> network. 
	```json copy
	[
		{
			"id":"ETH",
			"type": "coin",
			"blockchain": "ethereum",
			"chainId": 1
		},
		{
			"id": "USDC",
			"type": "token",
			"blockchain": "ethereum",
			"chainId": 1,
			"contract": "0xa0b86991c6218b36c1d19d4a2e9eb0ce3606eb48"
		}		
	]
	```
- **Account**: The Ledger wallet address on a <Glossary word="cryptocurrency"></Glossary> network. A user can have several accounts on the same <Glossary word="blockchain"></Glossary> (several <Glossary word="ethereum"></Glossary> addresses for instance)

### 9. **Creating a Manifest File**

A <Glossary word="manifest"></Glossary> file is necessary to test your <Glossary word="Live App"></Glossary> inside <Glossary word="Ledger Live"></Glossary>. Create a `manifest.json` file at the root of your project. Here is a sample <Glossary word="manifest"></Glossary> you can use:

```json
{
  "$schema": "https://live-app-catalog.ledger.com/schema.json",
  "id": "SampleLiveApp",
  "name": "SampleLiveApp",
  "url": "http://localhost:3000/",
  "homepageUrl": "http://localhost:3000/",
  "platform": ["ios", "android", "desktop"],
  "apiVersion": "^2.0.0",
  "manifestVersion": "2",
  "branch": "stable",
  "categories": ["NFT", "Swap", "YourAppCategory"],
  "currencies": "*",
  "content": {
    "shortDescription": {
      "en": "Desc"
    },
    "description": {
      "en": "Desc"
    }
  },
  "permissions": [
    "account.list",
    "account.receive",
    "account.request",
    "currency.list",
    "message.sign",
    "transaction.sign",
    "transaction.signAndBroadcast",
    "wallet.capabilities",
    "wallet.info",
    "wallet.userId"
  ],
  "domains": ["http://", "https://"],
  "visibility": "complete"
}
```

  For more informations regarding the setting up of a <Glossary word="manifest"></Glossary>, please refer to
  [the manifest page](../../appendix/manifest)
</Callout>

### 10. **Importing Manifest in Ledger Live**

Ensure <Glossary word="Ledger Live"></Glossary> Desktop is installed on your computer. Enable <Glossary word="Developer mode"></Glossary> in <Glossary word="Ledger Live"></Glossary> by navigating to **Settings** -> **About**, and clicking ten times on the <Glossary word="Ledger Live"></Glossary> version. This will reveal a new **Developer** section in the settings menu.

- Enable **platform dev tools**.
- Click on **Browse** next to **Add a local app** and select the <Glossary word="manifest"></Glossary> you created.

Now you'll see a new row in the menu with the name of your <Glossary word="Live App"></Glossary>. You can now open and test your <Glossary word="Live App"></Glossary> within <Glossary word="Ledger Live"></Glossary>.

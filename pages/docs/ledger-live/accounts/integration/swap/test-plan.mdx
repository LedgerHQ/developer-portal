# Manual tests

## Basic tests

Before you start testing the swap functionalities in deep, you need to make sure that the coin is whitelisted and well configured in the backends (check
How to :: Update SwapLiveApp ), so it appears in the Swap Live App. Also, you need to test if the Swap partners, like CIC and Changelly, are mapped correctly.

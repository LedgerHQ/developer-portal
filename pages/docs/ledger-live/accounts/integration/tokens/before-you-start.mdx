# Before you start

Before starting the development of tokens, ensure blockchain is already integrated and you have access to all the necessary repositories :

| GitHub repository                                      | Description                                                                                               |
| ------------------------------------------------------ | --------------------------------------------------------------------------------------------------------- |
| [Ledger Live](https://github.com/LedgerHQ/ledger-live) | Software Wallet on Desktop, Mobile and CLI.                                                               |
| Device app                                             | This repository contains the source code for the device app, which enables secure signing of transactions |

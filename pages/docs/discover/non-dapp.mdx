# How to add a Web3 App to Discover?

To integrate a **non-DApp** (not EVM based), you need to:

- Read the [submission guidelines](./guidelines)
- Use the [Wallet API](https://wallet.api.live.ledger.com/)
- Write a manifest as explained in the [Wallet API documentation](https://wallet.api.live.ledger.com/appendix/manifest)

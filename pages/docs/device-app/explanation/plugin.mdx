# Ethereum plugins

Plugins display specific smart contracts information to the users to avoid <Glossary word="blind signing"></Glossary>. They work hand-in-hand with the <Glossary word="Ethereum"></Glossary> App.

The <Glossary word="Ethereum"></Glossary> App handles parsing, signing, screen display etc. The only thing your plugin needs to do is:

1. Extract the relevant information from the data.
2. Send the string to be displayed back to the <Glossary word="Ethereum"></Glossary> App.

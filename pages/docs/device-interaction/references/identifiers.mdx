# Device and transport identifiers

In case of a custom integration (not using the DMK or LedgerJS), use the following identifiers to enable USB and BLE communication.

## Ledger Nano S Plus

- **USB Identifiers**:
  - `usbProductId`: `0x50`
  - `bootloaderUsbProductId`: `0x0005`

## Ledger Nano X

- **USB Identifiers**:
  - `usbProductId`: `0x40`
  - `bootloaderUsbProductId`: `0x0004`
- **Bluetooth Identifiers**:
  - `serviceUuid`: `13d63400-2c97-0004-0000-4c6564676572`
  - `notifyUuid`: `13d63400-2c97-0004-0001-4c6564676572`
  - `writeUuid`: `13d63400-2c97-0004-0002-4c6564676572`
  - `writeCmdUuid`: `13d63400-2c97-0004-0003-4c6564676572`

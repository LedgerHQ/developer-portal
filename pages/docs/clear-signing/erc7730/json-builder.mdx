# ERC-7730 JSON Builder for Smart contract clear signing

You can use the [ERC7730 JSON builder](https://get-clear-signed.ledger.com/) to easily create a compliant JSON file and visualize how the clear signing will like on a device.
It will create a ERC-7730 compliant JSON file that you can submit [following this process](./process). In this page, we take Aave as an example.

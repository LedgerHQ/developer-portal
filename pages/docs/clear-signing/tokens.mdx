# ERC-721, ERC-1155 and ERC-20 tokens

Clear signing for ERC-721, ERC-1155 and ERC-20 tokens is automatically handled by the Ethereum application if the specific token is in [Ledger supported coins and tokens list](https://www.ledger.com/supported-crypto-assets).

To learn more about token addition, [read this documentation](../ledger-live/accounts/integration/tokens/evm-chains-tokens).

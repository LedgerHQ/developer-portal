# Tool set to help your development

- Speculos emulates a device so that you can test your app
- The PRINTF macro debugging feature is supported by the firmware
- The PIN bypass helps you avoid entering the PIN each time you install an unsigned app
- Ledgerctl is a tool to get info about your devices and uninstall apps
